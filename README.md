# Excel Dashboard Project

## Overview

This project is an interactive Excel Dashboard created using raw dataset analysis, data cleaning, visualization, and dashboarding techniques.

The project demonstrates practical Excel skills useful in data analysis and business intelligence.

---

## Files Included

### 1. Excel Project Dataset.xlsx

Raw and messy dataset used for analysis.

### 2. Excel Project.xlsx

Final interactive dashboard project with charts, KPIs, filters, and visual analysis.

---

## Features

* Data Cleaning
* Pivot Tables
* Interactive Dashboard
* Charts and Graphs
* KPI Analysis
* Slicers and Filters
* Data Visualization

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting
* Slicers
* Data Cleaning Techniques
